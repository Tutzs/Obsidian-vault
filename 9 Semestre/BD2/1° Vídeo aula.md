O que é banco de dados? Uma coleção organizada de dados que tem alguma relação entre si de forma a criar sentido e possibilitar futuras consultas a esses dados

Precisamos gerir esses dados, 

Sistema de Gerenciamento de Banco de Dados, programas que ajudam a gerenciar os banco de dados

São programas que rodam no computador que precisam ter uma capacidade de armazenamento para gerir o banco de dados

Pode ser no seu computador (local) ou um banco de dados na nuvem

MYsql, Oracle, PostgreSQL, MongoDB

SQL, NoSQL

Relacionais: em tabelas
Priorizam: consistencia, atomicidade, não redundancia

Não relacionais: em documentos
Priorizam: flexibilidade, escalabilidade e rapidez na resposta as consultas feitas
