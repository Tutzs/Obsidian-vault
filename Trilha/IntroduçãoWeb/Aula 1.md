Protocolo HTTP (Hyper Text Tranport Protocol)
São regras para uma forma de conversa entre duas máquinas, permite transferir hiper-texto de um lado a outro

GET - busca um recurso para leitura
PUT - atualiza um recurso
PATCH - atualiza parcialmente um dado
DELETE - remove um dado
POST - cria um novo dado
HEAD - busca apenas o header de um recurso/dado

HTML linguagem de marcação 
contem links q conectam paginas entre si
CSS linguagem de estilização
JavaScript



tags em block-level ou inline

block-level (nivel de bloco) ocupa toda a largura de seu elemento pai (container) criando assim um "bloco"

inline (em linha) geralmente é usado para demarcação de conteúdo de texto

### block level
- p, header, main, footer, section, article, aside, nav, div, h1, h6, video, figure, li, menu, ol, ul
### inline
- a, span, br, img, audio, small, strong, sub, sup, time, u

tag semanticas

header - cabeçalhos
article - para dar um significado de artigo
main - conteudo principal
p - indicar paragrafo
footer - rodape da pagina
aside - conteudo extra, como barra lateral
nav - secao com links de navegação
ol - lista de item ordenado
ul - lista de item desordenado
li - representa um item que faz parte de uma lista
